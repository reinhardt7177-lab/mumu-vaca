# Neo-OutRun Asset Slots (CC0)

`games/social-racing-outrun/src/assets/asset-manifest.js`에서 아래 파일 경로를 사용합니다.

외부 에셋을 넣지 않아도 게임은 `fallback-textures.js`로 기본 그래픽을 자동 생성해 실행됩니다.

## 폴더 구조

- `bg/mountains.png`
- `bg/forest.png`
- `bg/roadside.png`
- `cars/player.png`
- `cars/rival-1.png`
- `cars/rival-2.png`
- `cars/rival-3.png`
- `cars/traffic-1.png`

- `fx/spark.png`
- `sfx/boost.ogg`
- `sfx/hit.ogg`
- `sfx/checkpoint.ogg`
- `sfx/select.ogg`

## 권장 규격

- 배경(`bg/*`): `512x256` 또는 `1024x512`
- 차량(`cars/*`): 탑뷰 기준 `40x24` 또는 `80x48` (픽셀 아트)
- UI 패널(`ui/*`): `320x96`, `320x220`
- 스파크(`fx/spark.png`): `12x12` 또는 `24x24`
